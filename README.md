# рҹҡҖ Installation

Clone the repository:

```bash
git clone https://github.com/PranjitaModi/SocialSync.git
```

Navigate to the project directory:

```bash
cd SocialSync
```

Install backend dependencies:

```bash
cd server
npm install
```

Install frontend dependencies:

```bash
cd ../client
npm install
```
